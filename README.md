# EduTJ — Платформаи Омӯзишии Тоҷикистон

## 📁 Сохтори файлҳо
```
lms/
├── app.py              ← Flask сервер (асосӣ)
├── lms.db              ← SQLite маълумотгоҳ (худ месозад)
├── requirements.txt    ← Китобхонаҳо
├── static/
│   ├── css/style.css   ← Стилҳо
│   └── uploads/
│       ├── videos/     ← Видеоҳои зерорфоришуда
│       └── images/     ← Тасвирҳо
└── templates/
    ├── index.html      ← Саҳифаи асосӣ
    ├── login.html      ← Дохилшавӣ
    ├── admin.html      ← Панели администратор
    ├── pupil.html      ← Кабинети донишҷӯ
    ├── test.html       ← Тестҳо
    ├── chat.html       ← Чат
    ├── interpretator.html ← Интерпретатори Python
    ├── sertifikat.html ← Сертификат
    ├── reiting.html    ← Рейтинг
    └── zadacha.html    ← Вазифаҳо
```

## 🚀 Оғоз кардан

### 1. Насб кардани китобхонаҳо
```bash
pip install flask werkzeug
```
ё
```bash
pip install -r requirements.txt
```

### 2. Иҷро кардан
```bash
python app.py
```

### 3. Кушодан дар браузер
```
http://localhost:5000
```
